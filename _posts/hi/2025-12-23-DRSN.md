## 2. Soft Thresholding

**Soft Thresholding ‡§ï‡§à signal denoising algorithms ‡§ï‡§æ core step ‡§π‡•à‡•§ ‡§Ø‡§π ‡§â‡§® features ‡§ï‡•ã delete ‡§ï‡§∞ ‡§¶‡•á‡§§‡§æ ‡§π‡•à ‡§ú‡§ø‡§®‡§ï‡•Ä absolute value ‡§è‡§ï fix threshold ‡§∏‡•á ‡§ï‡§Æ ‡§π‡•ã‡§§‡•Ä ‡§π‡•à, ‡§î‡§∞ ‡§ú‡§ø‡§®‡§ï‡•Ä value ‡§á‡§∏ threshold ‡§∏‡•á ‡§ú‡•ç‡§Ø‡§æ‡§¶‡§æ ‡§π‡•ã‡§§‡•Ä ‡§π‡•à, ‡§â‡§®‡•ç‡§π‡•á‡§Ç zero ‡§ï‡•Ä direction ‡§Æ‡•á‡§Ç shrink ‡§ï‡§∞ ‡§¶‡•á‡§§‡§æ ‡§π‡•à‡•§** ‡§á‡§∏‡•á ‡§®‡•Ä‡§ö‡•á ‡§¶‡§ø‡§è ‡§ó‡§è formula ‡§∏‡•á implement ‡§ï‡§ø‡§Ø‡§æ ‡§ú‡§æ ‡§∏‡§ï‡§§‡§æ ‡§π‡•à:

$$

$$

**Soft Thresholding** ‡§ï‡§æ output, input ‡§ï‡•á respect ‡§Æ‡•á‡§Ç (derivative) ‡§á‡§∏ ‡§™‡•ç‡§∞‡§ï‡§æ‡§∞ ‡§π‡•à:

$$
\frac{\partial y}{\partial x} = \begin{cases}

0 & -\tau \le x \le \tau \\

\end{cases}
$$

‡§ú‡•à‡§∏‡§æ ‡§ï‡§ø ‡§ä‡§™‡§∞ ‡§¶‡•á‡§ñ‡§æ ‡§ú‡§æ ‡§∏‡§ï‡§§‡§æ ‡§π‡•à, **Soft Thresholding** ‡§ï‡§æ derivative ‡§Ø‡§æ ‡§§‡•ã 1 ‡§π‡•à ‡§Ø‡§æ 0‡•§ ‡§Ø‡§π property **ReLU activation function** ‡§ï‡•á ‡§ú‡•à‡§∏‡•Ä ‡§π‡•Ä ‡§π‡•à‡•§ ‡§á‡§∏‡§≤‡§ø‡§è, **Soft Thresholding** ‡§≠‡•Ä **Deep Learning algorithms** ‡§Æ‡•á‡§Ç **gradient vanishing** ‡§î‡§∞ **gradient exploding** ‡§ï‡•á risk ‡§ï‡•ã ‡§ï‡§Æ ‡§ï‡§∞‡§®‡•á ‡§Æ‡•á‡§Ç ‡§∏‡§ï‡•ç‡§∑‡§Æ ‡§π‡•à‡•§

**Soft Thresholding function ‡§Æ‡•á‡§Ç, threshold set ‡§ï‡§∞‡§§‡•á ‡§∏‡§Æ‡§Ø ‡§¶‡•ã conditions ‡§ï‡§æ ‡§π‡•ã‡§®‡§æ ‡•õ‡§∞‡•Ç‡§∞‡•Ä ‡§π‡•à: ‡§™‡§π‡§≤‡§æ, threshold ‡§è‡§ï positive number ‡§π‡•ã‡§®‡§æ ‡§ö‡§æ‡§π‡§ø‡§è; ‡§¶‡•Ç‡§∏‡§∞‡§æ, threshold input signal ‡§ï‡•Ä maximum value ‡§∏‡•á ‡§¨‡•ú‡§æ ‡§®‡§π‡•Ä‡§Ç ‡§π‡•ã‡§®‡§æ ‡§ö‡§æ‡§π‡§ø‡§è, ‡§µ‡§∞‡§®‡§æ output ‡§™‡•Ç‡§∞‡§æ zero ‡§π‡•ã ‡§ú‡§æ‡§è‡§ó‡§æ‡•§**

**‡§á‡§∏‡§ï‡•á ‡§∏‡§æ‡§• ‡§π‡•Ä, threshold ‡§ï‡•ã ‡§§‡•Ä‡§∏‡§∞‡•Ä condition ‡§≠‡•Ä ‡§™‡•Ç‡§∞‡•Ä ‡§ï‡§∞‡§®‡•Ä ‡§ö‡§æ‡§π‡§ø‡§è: ‡§π‡§∞ sample ‡§ï‡§æ ‡§â‡§∏‡§ï‡•á ‡§Ö‡§™‡§®‡•á noise content ‡§ï‡•á base ‡§™‡§∞ ‡§Ö‡§™‡§®‡§æ ‡§è‡§ï independent threshold ‡§π‡•ã‡§®‡§æ ‡§ö‡§æ‡§π‡§ø‡§è‡•§**

‡§ê‡§∏‡§æ ‡§á‡§∏‡§≤‡§ø‡§è ‡§π‡•à ‡§ï‡•ç‡§Ø‡•ã‡§Ç‡§ï‡§ø ‡§ï‡§à ‡§¨‡§æ‡§∞ samples ‡§Æ‡•á‡§Ç noise ‡§ï‡§æ amount ‡§Ö‡§≤‡§ó-‡§Ö‡§≤‡§ó ‡§π‡•ã‡§§‡§æ ‡§π‡•à‡•§ Example ‡§ï‡•á ‡§≤‡§ø‡§è, ‡§Ö‡§ï‡•ç‡§∏‡§∞ ‡§ê‡§∏‡§æ ‡§π‡•ã‡§§‡§æ ‡§π‡•à ‡§ï‡§ø ‡§è‡§ï ‡§π‡•Ä sample set ‡§Æ‡•á‡§Ç Sample A ‡§Æ‡•á‡§Ç ‡§ï‡§Æ noise ‡§π‡•à ‡§î‡§∞ Sample B ‡§Æ‡•á‡§Ç ‡§ú‡•ç‡§Ø‡§æ‡§¶‡§æ‡•§ ‡§ê‡§∏‡•á ‡§Æ‡•á‡§Ç, denoising algorithm ‡§Æ‡•á‡§Ç soft thresholding ‡§ï‡§∞‡§§‡•á ‡§∏‡§Æ‡§Ø, Sample A ‡§ï‡•á ‡§≤‡§ø‡§è ‡§¨‡•ú‡§æ threshold ‡§î‡§∞ Sample B ‡§ï‡•á ‡§≤‡§ø‡§è ‡§õ‡•ã‡§ü‡§æ threshold use ‡§ï‡§ø‡§Ø‡§æ ‡§ú‡§æ‡§®‡§æ ‡§ö‡§æ‡§π‡§ø‡§è‡•§ Deep Neural Network ‡§Æ‡•á‡§Ç, ‡§≠‡§≤‡•á ‡§π‡•Ä ‡§á‡§® features ‡§î‡§∞ thresholds ‡§ï‡§æ clear physical meaning ‡§® ‡§π‡•ã, ‡§≤‡•á‡§ï‡§ø‡§® basic logic ‡§µ‡§π‡•Ä ‡§∞‡§π‡§§‡§æ ‡§π‡•à‡•§ ‡§Ø‡§æ‡§®‡•Ä, ‡§π‡§∞ sample ‡§ï‡§æ ‡§â‡§∏‡§ï‡•á noise content ‡§ï‡•á ‡§π‡§ø‡§∏‡§æ‡§¨ ‡§∏‡•á ‡§Ö‡§™‡§®‡§æ ‡§Ö‡§≤‡§ó threshold ‡§π‡•ã‡§®‡§æ ‡§ö‡§æ‡§π‡§ø‡§è‡•§
